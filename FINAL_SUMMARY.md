## 简历建议

### ✅ 应该这样写

"**AutoDev - 多 Agent 协作代码生成系统**

设计并实现了基于多 Agent 协作的代码生成系统，通过 5 个专业 Agent 的协作，实现从需求分析到代码实现、测试、审查的完整自动化流程。

**技术亮点**：
- 实现了 5 Agent 协作工作流，生成 Clean Architecture 架构的代码（平均 9.2 个模块化文件）

**实验结果**：
- 在 5 个不同复杂度任务上对比 Single LLM baseline
